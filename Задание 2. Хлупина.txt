Чтобы понять, что такое DNS (Domain Name System) для начала нужно разобраться, как это определение переводится. Domain Name System переводится как Система Доменных Имен или Система Имен Доменов.
Теперь подробнее, основная задача такого сервера – хранение информации, которую в любой момент может запросить пользователь через браузер. Какую информацию хранит DNS сервер? Обычно это IP-адреса сайтов. Например: пользователь в адресной строке запрашивает сайт youtube.com устройство (компьютер или смартфон) запрашивает IP-адрес этого сайта у DNS, сервер даёт обратную связь устройству и сайт открывается.
Далее разберемся, что такое HTTP и HTTPS протоколы и в чем их различие. HTTP ( Hypertext Transfer Protocol) – протокол передачи гипертекста.
HTTPS (Hypertext Transfer Protocol Secure) – безопасный протокол передачи гипертекста.
Видим, что основная разница заключается в букве S,  данная буква означает безопасность. То есть протокол HTTPS это тот же протокол HTTP, но с защитой данных. Сейчас чаще всего используется именно протокол HTTPS.
Для чего нужны протоколы HTTP и HTTPS? – При передачи данных по принципу «сервер – пользователь и обратно» все действия протоколируются.
Протокол – это запись в определенном формате, фиксирующая произошедшие или происходящие события.
Версии протокола HTTP и их различия:
Версия 0.9 - данная версия была разработана, чтобы навести порядок во взаимодействии клиента и сервера. Появилось разделение функций между ними.
Версия 1.0 - документ данной версии стал основой для реализации приложений и компонентов, пользователи могут идентифицировать свои приложения при передаче HTTP сообщений.
Версия 1.1 - данная версия позволила за одно соединение отправлять несколько HTTP запросов и получать столько же HTTP ответов. Второй особенностью стало то, что теперь клиент при установке соединения с сервером должен обязательно посылать имя хоста в поле HTTP заголовка. Ну и были исправлены мелкие недоработки в дизайне, исправлены опечатки.
Версия 2.0 - в данной версии поддерживается объединение запросов и соответственно появится приоритет запросов.
